My Projects

Academix

Academix is a website I built as part of a group project for a class, where the main objective was to develop a functional application. I led the team, conceptualizing the idea, assigning tasks, and assisting teammates. Academix is a study and class management tool designed to enhance study efficiency by tracking classes, homework, and mastery of topics.

Markov Chain Story Generator

One of my earliest and most memorable projects, the Markov Chain Story Generator, introduced me to algorithmic text generation. Users input a text, and the program uses Markov chains to generate coherent stories. This project enhanced my understanding of Java and the practical applications of algorithms.

Closet Lab

Closet Lab is a mobile app developed in a group project to help users organize and visualize their outfits. The app supports uploading photos of clothing, creating outfits, and planning with a calendar, alongside donation reminders for unused items. We followed real software development practices, including creating software requirement documents, system design documents, unit testing, and system testing. I contributed to both the React Native frontend and the Python/MongoDB backend.
